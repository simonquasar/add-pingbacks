Manually add a Pingback to a post or page.

== Description ==

Adds an options page in the Comments menu. 
You select on which Post/Page the referring Link should appear, add the Site name and URL, then paste the referring part.

== Installation ==

1. Download the zip file and extract the `add-pingbacks` directory.
2. Upload this directory inside your `/wp-content/plugins/` directory.
3. Browse to the `Add Comments` page on the Options menu.
